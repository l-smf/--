# 💪 Workout Schedule

جدول تمرين أسبوعي تفاعلي للمبتدئ — ٥ أيام في الأسبوع.

## المميزات

- ٥ أيام تمرين (Push / Pull / Legs / Upper / Lower)
- كل عضلة تتمرن مرتين في الأسبوع
- تتبع كل سيت بشكل مستقل
- عرض ملخص اليوم بضغطة واحدة
- يحفظ تقدمك تلقائياً
- يشتغل بدون إنترنت (PWA)
- يفتح يوم اليوم تلقائياً

## كيف تشغّله على GitHub Pages

1. ارفع الملفات على GitHub repository
2. روح **Settings → Pages**
3. اختار `main` branch → Save
4. افتح الرابط على جوالك في Safari أو Chrome
5. اضغط "إضافة إلى الشاشة الرئيسية"

## الملفات

```
index.html      ← التطبيق الرئيسي
manifest.json   ← إعدادات PWA
sw.js           ← Service Worker للعمل offline
icons/          ← أيقونات التطبيق
```
